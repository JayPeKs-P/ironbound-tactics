Detailed Description

Stores all information regarding the transformation (e.g. position, rotation) of an entity.

Transforms are part of the 'Scene Graph'. They are nodes in tree structure. Transforms may have many children but only one a single parent. If the parent transform is changed in any way, the effect is applied to all its child nodes. Transform is a Component, therefore the constructor is private and can only be called by ComponentManager.
