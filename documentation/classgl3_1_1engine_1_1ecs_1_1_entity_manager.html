Detailed Description

The class that handles the lifetime of gl3::engine::ecs::Entity objects.

An object of this class is member of @ref gl3::engine::Game. Use it to create new entities. It adds a unique_ptr of a new entity to a container and maps it to its unique ID of type gl3::engine::ecs::guid_t.
