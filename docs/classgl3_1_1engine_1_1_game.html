Detailed Description

Core game class, that manages the main structures the main loop and manages core systems and ECS state.

This class initializes the core components (context, ECS, rendering, events) and defines the core structure of a games lifecycle and update loop.

◆ run()

virtual void gl3::engine::Game::run ( )
virtual

Starts the main game loop.

Runs the game. Invokes all core events and alls start() function.

This function initializes the rendering system, the updater and pruner of the Scene Graph. It then calls the start() function, invokes all core events and enters the core loop via 'context.run()', invoking update() and draw() and the corresponding core events.

Lifecycle events triggered (in order):

  • onStartup
  • onAfterStartup
  • onBeforeUpdate → update() → onUpdate → draw() → renderer.draw() → updateDeltaTime → onAfterUpdate
  • onBeforeShutdown
  • onShutdown
