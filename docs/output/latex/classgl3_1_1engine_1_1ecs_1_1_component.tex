\doxysubsection{Detailed Description}
Abstract base class for all ECS components. 

Every \doxylink{classgl3_1_1engine_1_1ecs_1_1_component}{Component} has an owner. It also stores the guid\+\_\+t ID of the entity they belong to. The lifetime of all components should be managed by an instance of \doxylink{classgl3_1_1engine_1_1ecs_1_1_component_manager}{Component\+Manager}. \doxylink{classgl3_1_1engine_1_1ecs_1_1_component}{Component} has two friend classes\+: \doxylink{classgl3_1_1engine_1_1ecs_1_1_entity}{Entity} and \doxylink{classgl3_1_1engine_1_1ecs_1_1_component_manager}{Component\+Manager}.
