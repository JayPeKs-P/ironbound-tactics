\doxysubsection{Detailed Description}
Abstract base class for all ECS entities. 

\doxylink{classgl3_1_1engine_1_1ecs_1_1_entity}{Entity} primarily consists of its ID. It has a reference to \doxylink{classgl3_1_1engine_1_1ecs_1_1_component_manager}{gl3\+::engine\+::ecs\+::\+Component\+Manager} so it can call its utility functions. \doxylink{classgl3_1_1engine_1_1ecs_1_1_entity}{Entity} is declared final to prevent inheritance. To remove an entity, it is first flagged for deletion and then purged later during the update cycle by \doxylink{classgl3_1_1engine_1_1ecs_1_1_entity_manager}{gl3\+::engine\+::ecs\+::\+Entity\+Manager}.
