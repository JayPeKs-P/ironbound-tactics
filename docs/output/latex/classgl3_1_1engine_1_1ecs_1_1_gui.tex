\doxysubsection{Detailed Description}
Abstract base class for all GUI scenes. 

A GUI scene has access to the ECS context (\doxylink{classgl3_1_1engine_1_1_game}{engine\+::\+Game} object).

All GUI scenes should inherit from this class.
