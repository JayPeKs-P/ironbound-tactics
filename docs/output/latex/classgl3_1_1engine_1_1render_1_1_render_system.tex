\doxysubsection{Detailed Description}
Handles Open\+GL rendering of 2D models with support for instancing. 

This system draws all entities with a \doxylink{struct_model2_d}{Model2D} and optionally \doxylink{struct_instance_buffer}{Instance\+Buffer} component using either instanced or non-\/instanced rendering. All member functions and variables are set to private. \textquotesingle{}\doxylink{classgl3_1_1engine_1_1_game}{Game}\textquotesingle{} is registered as friend class.
