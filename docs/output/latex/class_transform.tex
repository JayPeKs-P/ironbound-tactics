\doxysubsection{Detailed Description}
Stores all information regarding the transformation (e.\+g. position, rotation) of an entity. 

Transforms are part of the \textquotesingle{}Scene Graph\textquotesingle{}. They are nodes in tree structure. Transforms may have many children but only one a single parent. If the parent transform is changed in any way, the effect is applied to all its child nodes. \doxylink{class_transform}{Transform} is a \doxylink{class_component}{Component}, therefore the constructor is private and can only be called by \doxylink{class_component_manager}{Component\+Manager}.
