\doxysubsection{Detailed Description}
The class that handles the lifetime of \doxylink{classgl3_1_1engine_1_1ecs_1_1_entity}{gl3\+::engine\+::ecs\+::\+Entity} objects. 

An object of this class is member of @ref \doxylink{classgl3_1_1engine_1_1_game}{gl3\+::engine\+::\+Game}. Use it to create new entities. It adds a unique\+\_\+ptr of a new entity to a container and maps it to its unique ID of type gl3\+::engine\+::ecs\+::guid\+\_\+t.
